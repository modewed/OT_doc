Class: DOMDocumentType

Each Document has a doctype attribute whose value is either null or a DocumentType object. The DocumentType interface in the DOM Core provides an interface to the list of entities that are defined for the document, and little else because the effect of namespaces and the various XML schema efforts on DTD representation are not clearly understood as of this writing.

The DOM Level 2 doesn't support editing DocumentType nodes.

Instance Methods

GetEntities( )

A NamedNodeMap containing the general entities, both external and internal, declared in the DTD.

GetInternalSubset( )

Returns the internal DTD as a string (without the delimiting square brackets), or null if there is none.

GetName( )

The name of DTD; i.e., the name immediately following the DOCTYPE keyword.

GetNotations( )

A NamedNodeMap containing the notations declared in the DTD.

GetPublicId( )

Returns the public identifier of the external subset.

GetSystemId( )

Returns the system identifier of the external subset.

Instance Methods

GetEntities

DOMNamedNodeMap GetEntities()

A NamedNodeMap containing the general entities, both external and internal, declared in the DTD. Parameter entities are not contained. Duplicates are discarded. For example in: 

<!DOCTYPE ex SYSTEM "ex.dtd" [
  <!ENTITY foo "foo">
  <!ENTITY bar "bar">
  <!ENTITY bar "bar2">
  <!ENTITY % baz "baz"> 
]>

the interface provides access to foo and the first declaration of bar but not the second declaration of bar or baz. Every node in this map also implements the Entity interface. The DOM Level 1 does not support editing entities, therefore entities cannot be altered in any way.

Returns:

A DOMNamedNodeMap of the entities.

GetInternalSubset

String GetInternalSubset()

Returns the internal DTD as a string (without the delimiting square brackets), or undefined if there is none.

Returns:

The internal DTD.

GetName

String GetName()

The name of DTD; i.e., the name immediately following the DOCTYPE keyword.

Returns:

The name of the DTD.

GetNotations

DOMNamedNodeMap GetNotations()

A NamedNodeMap containing the notations declared in the DTD. Duplicates are discarded. Every node in this map also implements the Notation interface. The DOM Level 1 does not support editing notations, therefore notations cannot be altered in any way.

Returns:

A DOMNamedNodeMap of the notations.

GetPublicId

String GetPublicId()

Returns the public identifier of the external subset.

Returns:

The public identifier.

GetSystemId

String GetSystemId()

Returns the system identifier of the external subset.

Returns:

The system identifier.
