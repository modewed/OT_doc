Class: DOMNodeList

The DOMNodeList interface provides the abstraction of an ordered collection of nodes, without defining or constraining how this collection is implemented. The items in the NodeList are accessible via an integral index, starting from 0.

Instance Methods

GetLength( )

The number of nodes in the list. The range of valid child node indices is 0 to length-1 inclusive.

Item( Integer arg1 )

Returns the indexth item in the collection. If index is greater than or equal to the number of nodes in the list, this returns null.
