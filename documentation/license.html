Class: License

Important Note: The License built-ins are supposed to be used internally only, that is, by the License Subsystem in the Kernel module which provides hooks for modules requiring licenses. They are not meant to be used by 3rd party developers. At present the licensing functionality depends on SafeNet. However, that may change in the future.

The License package provides services for retrieving and managing licenses for the Content Server and modules that require separate licenses. The major functionalities offered in the License Package are the following:

  • ApplyCoreLicense() Loads and validate a license file and sets it as the Content Server core license.
  • ApplyModuleLicense() Loads and validates a module license file and sets it as a module license.
  • GenerateFingerprint() Generates a fingerprint string for the installed system.
  • GenerateReportHash() Returns a hash generated from the license data.
  • GetCoreLicenseInfo() Gets information from the Content Server core license.
  • GetLicensedApplications() Returns a list of licensed applications.
  • GetLicenseFeatureName() Returns the feature name for the license.
  • GetLicenseInfo() Gets the information from a license content string or a license file path.
  • GetLicenseLogString() Returns a string containing the license info for each license.
  • GetModuleLicenseInfo() Gets information from a Content Server module license.
  • GetNumActiveUsers() Returns the number of active users in the system.
  • ValidateCoreLicense() Validates the Content Server core license.
  • ValidateModuleLicense() Validates a Content Server module license.
